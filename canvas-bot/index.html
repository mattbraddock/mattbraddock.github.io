Canvas Bot

The bot (@braddock_canvas) does two things:

  • Tweets out new announcements
  • Replies to users with upcoming assignments

To get upcoming assignments, you must tag the bot and include two things: the course and the type of assignment.
Note: options are not case sensitive.

Options for courses include:

  • Geometry GT (which can be shortened to Geo GT)
  • SAT Prep

Options for assignments include:

  • Assignments (alternatives that can be used are classwork, CW, homework, HW, learning tasks)
  • Assessments (alternatives that can be used are quiz, test, exam)

Example: @braddock_canvas geo gt test
